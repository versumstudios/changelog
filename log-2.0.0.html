2.0.0 - December 19th, 2022

New NFT collections

We've added the following popular Tezos NFT collections to Versum:

  • hicetnunc
  • 8bidou - 8x8, 24x24x1, 24x24x3
  • Neonz
  • Distributed Consciousness
  • Verses / Distributed Consciousness
  • Tezzards
  • Ottez
  • Ziggurats
  • Project GOGOs

Head to Collections to see them. If you are following creators that have minted NFTs for these collections, or boards that contain some, they will appear in your personal feed under Explore.

Full support for creator royalties for these tokens is guaranteed when tokens are listed on Versum marketplace. This is done with an on-chain contract call that registers the royalties data using informartion indexed from the original token metadata. This data is then made available via on-chain views on the Versum Royalties Adaptor contract KT1NffZ1mqqcXrwYY3ZNaAYxhYkyiDvvTZ3C

We will continue to add more collections, please let us know if there are any in particular that you'd like to see on Discord or Twitter.

Support for other marketplaces' listings

Versum now aggregates & displays sales listings from other NFT marketplaces. This gives you the convenience of seeing a wider choice of prices for NFTs and it lets you collect at the best price. The current marketplaces supported are:

  • objkt.com - swaps and offers
  • TEIA - swaps
  • 8BIDOU - swaps

Improved support for tokens requiring Versum Verfied Owners

When minting on Versum, creators have always had the option to restrict collection by non-verified users. If you tried to purchase one of these tokens as an unverified user, you would have received an error.

The user experience for this has been improved, and the token's Info tab now shows this information in the "Verfied Owners Only" field. In addition, you will see a related message on any "buy" buttons.
